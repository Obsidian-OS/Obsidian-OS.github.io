Install ObsidianOS

As ObsidianOS is image-based, the installation is quite diffrent than arch.
Premade archisos are not ready yet, so you must either:

  1. Create your own archisos, or
  2. Install mkobsidiansfs and obsidianctl to install from an arch host.
Then you will need to follow the installation guide.
