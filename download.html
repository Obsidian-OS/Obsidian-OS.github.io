Install ObsidianOS

As ObsidianOS is image-based, the installation is quite diffrent than Arch.

  1. Create your own ISO or get one from the button below
  2. Install mkobsidiansfs and obsidianctl to install ObsidianOS on a disk from an Arch host.
  3. NEW! From an existing arch host, run 
    sudo bash -c "$(curl -fsSL https://ba.sh/ARbs)"
Then you will need to follow the installation guide.
